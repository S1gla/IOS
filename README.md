# Wizytówka Portfolio

Projekt wizytówki/portfolio w React Native i Expo Router

## Uruchomienie

```bash
npm install
npx expo start
```

## Co zawiera

* ekran Home,
* ekran Projekty,
* szczegóły projektu,
* dodawanie i usuwanie projektu,
* ekran Kontakt z formularzem i walidacją,
* ekran O mnie z edycją profilu,
* dolną nawigację,
* Context API,
* AsyncStorage do zapisu profilu i projektów.
